# macOS Mouse Hook

A resident app that modifies mouse behavior on macOS.

- Right-click fires on button release
- Drag upward while holding right-click to open Mission Control
- Drag rightward while holding right-click to switch to the right desktop
- Drag leftward while holding right-click to switch to the left desktop
- Scroll direction matches wheel rotation direction
- Scroll amount changed from acceleration-based to a fixed amount per tick

> [!NOTE]
> Accessibility permission is required.
> You will be prompted to grant permission on first launch.
